## 🚀 Cómo ejecutar (para el profe)

### Opción A — VS Code + Live Server (recomendado)

1. Abrir esta carpeta del proyecto (raíz) en **VS Code**.
2. Abrir el archivo [index.html](index.html).
3. Click derecho → **Open with Live Server** (o botón **Go Live**).
4. Se abrirá el menú principal en el navegador. Desde ahí navega a:
   - [Frogger/index.html](Frogger/index.html)
   - [Space_Invaders/index.html](Space_Invaders/index.html)
   - [Tapper/index.html](Tapper/index.html)
   - [Snowy_Sokobear/index.html](Snowy_Sokobear/index.html)

> Nota: usar servidor local evita errores típicos de carga de assets (imágenes/sonidos) que a veces fallan con `file:///`.

### Opción B — Abrir directo (puede fallar en algunos navegadores)

- Abrir [index.html](index.html) con doble click.
- Si aparece algún error de recursos (404 / imágenes que no cargan), usa la **Opción A**.
